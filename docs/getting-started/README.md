## Quick Start

See the main [README](../../README.md) for installation instructions.

## Prerequisites

- **Node.js** 18+
- **Python** 3.11+
- **Docker Desktop**
- **OpenAI API Key**

## Installation

```bash
# Install all dependencies
make install

# Start development environment
make dev

# Run all tests
make test
```
